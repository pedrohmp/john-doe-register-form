# John Doe e seu form de cadastro

O projeto foi separado em duas pastas:
- Client (React + TypeScript + Vite)
- Server (NodeJS + Typeorm + Express)
- Banco de dados (Postgres)

## Rodando o projeto
### Client
```bash
   cd client
   yarn
   yarn dev
```

### Server
```bash
   cd server
   yarn
   docker-compose up
   yarn dev
```
